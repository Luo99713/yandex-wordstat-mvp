Yandex Wordstat MVP v0.6 - 俄罗斯选品特征分析

用途：
1. 输入一个或多个俄语产品类目词，如 сумка、рюкзак、чехол для телефона。
2. 工具调用 Yandex Wordstat API，采集热门查询、相似查询、趋势、地区分布。
3. 输出一个 Excel 文件，并自动生成选品特征分析：颜色、材质、尺寸、风格、人群、场景、功能、价格/购买意图。

重要限制：
- Wordstat 不是销量数据，也不是客户评价数据。它只能证明“俄罗斯用户在 Yandex 上主动搜索了哪些词”。
- 因此工具输出的是“搜索需求线索/产品特征候选”，不能直接等同于“必定好卖”。
- 最终选品仍建议结合 WB/Ozon 的销量、价格、评价、广告、物流成本一起判断。

推荐使用流程：
第一轮：只输入产品大类词，例如：
сумка
рюкзак
чехол для телефона

勾选：TopRequests、Dynamics、Regions、生成选品特征分析。
返回词数先用 20 或 50，请求间隔 1.5-2 秒，避免 429。

第二轮：打开输出 Excel 的 NextSeeds sheet，把高频且有特征的词复制出来做下一轮种子词。
例如第一轮输入 сумка 后，可能得到 сумка через плечо、бежевая сумка、кожаная сумка 等。

Excel Sheet 说明：
README_汇总：运行配置和行数汇总。
FeatureAnalysis：自动汇总的产品特征候选，包含证据查询词。
Queries_Classified：每条查询被识别出的特征标签。
NextSeeds：建议用于第二轮扩展的种子词。
SeedBank：不同类型种子词的用途和预期。
TopQueries：Yandex 返回的热门查询原始数据。
Associations：Yandex 返回的相似查询/用户还搜了什么。
Dynamics：趋势变化。
Regions：地区分布。
Errors：错误记录。
Settings：本次运行设置。

种子词建议：
- 产品类目词：сумка、рюкзак、чехол для телефона、автомобильный пылесос
  预期得到：颜色、材质、款式、尺寸、人群、购买意图。
- 节日词：новый год、8 марта、день рождения、масленица、пасха、1 сентября
  预期得到：俄罗斯人重视的节日、礼物、祝福、家庭活动、开学等。
- 天气季节词：погода、мороз、снег、жара、море
  预期得到：季节性需求、保暖、防水、户外、海边、旅行等。
- 场景词：дача、школа、в машину、для кухни、путешествие
  预期得到：dacha、学校、车用、厨房、旅行等生活场景。

运行：
Windows 双击 run_windows.bat，或命令行运行：python yandex_wordstat_mvp.py
API Key 请填 AQ 开头的 secret key，不要填 aj 开头的 Key ID。
