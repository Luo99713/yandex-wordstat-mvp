# Release Notes - v0.5.1

## 本版本目标

解决前期版本中入口过多、弹窗选择后无响应、API设置保存不稳定、HTML展示混乱等问题。

## 主要变化

- 使用 Python 图形窗口作为主入口；
- `START.bat` 只负责启动窗口；
- `API_SETTINGS.bat` 独立保留；
- 运行日志在窗口中实时显示；
- API 设置保存后会回读校验；
- 保留普通分析、品牌分析、维度生成、多类目批量；
- 输出文件名加入类目名；
- 统一结果目录；
- 保留 `下单/搜索` 和 `搜索订单对比`。

## 已知边界

- API 输出质量取决于所使用模型；
- 新类目第一次必须生成词典；
- 批量分析只使用已有词典缓存；
- 不应提交 API Key、WB 原始数据和运行结果。

## 建议后续改进

- 将完整工具包作为 GitHub Release asset 管理；
- 为常用类目沉淀人工校准后的词典；
- 增加广告数据与搜索表联动分析；
- 增加采购成本、重量、物流费、售价和毛利的经营适配分。
